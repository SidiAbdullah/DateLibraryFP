# \# DateLibraryFP 📅

# 

# A C++ Date Utility Library that provides a collection of functions to work with dates, months, years, and calendars.

# 

# \---

# 

# \## 📌 Project Overview

# 

# This project is a lightweight date-handling library built in C++.  

# It focuses on understanding how date systems work internally, including leap years, calendar calculations, and date manipulations without relying heavily on built-in libraries.

# 

# \---

# 

# \## ⚙️ Features

# 

# \- Check if a year is a leap year

# \- Calculate number of days, hours, minutes, and seconds in a year

# \- Get number of days in each month

# \- Calculate passed and remaining days in a year

# \- Add days to a specific date

# \- Compare two dates

# \- Get day of the week for any date

# \- Print monthly calendar

# \- Print full yearly calendar

# 

# \---

# 

# \## 🧠 Concepts Used

# 

# \- Date algorithms (Zeller’s Congruence)

# \- Modular arithmetic

# \- Arrays for optimization

# \- Clean function-based design

# \- Basic performance improvements

#
